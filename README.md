# Strategy Labs Plugin Marketplace

Official plugin marketplace from [Strategy Labs](https://strategylabs.us) — boutique performance marketing tools for eCommerce and lead generation brands.

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| **brand-content-os** | Brand & Content Operating System — capture brand knowledge, enforce voice consistency, and generate on-brand content across paid ads, email, SMS, Shopify, and social channels. Includes Fast.io team sync and organized output directories. | 1.1.0 |
| **customer-driven-optimizations** | Analyze recorded customer phone calls to extract competitive intelligence, identify site optimization opportunities, map customer journeys, and generate a deployable HTML dashboard. Supports both audio files and pre-made transcripts. Integrates with Brand Content OS for brand context. | 1.0.0 |
| **local-business-marketing-os** | Full marketing operating system for local restaurants and hospitality businesses. Includes brand onboarding, review responses, email/SMS campaigns, weekly social content briefs, digital asset organization, and inbox management. | 1.0.0 |

## Quick Start

### 1. Add the Marketplace

In Claude Cowork or Claude Code, run:

```
/plugin marketplace add strategy-labs/strategy-labs-marketplace
```

### 2. Install a Plugin

```
/plugin install brand-content-os@strategy-labs-plugins
```

### 3. Start Using It

Once installed, run `/new-brand-setup` to create your brand profile — import an existing document, share a URL, or answer guided questions.
