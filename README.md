# Strategy Labs Plugin Marketplace

Official plugin marketplace from [Strategy Labs](https://strategylabs.us) — boutique performance marketing tools for eCommerce and lead generation brands.

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| **brand-content-os** | Brand & Content Operating System — capture brand knowledge, enforce voice consistency, and generate on-brand content across paid ads, email, SMS, Shopify, and social channels. | 1.0.0 |

## Quick Start

### 1. Add the Marketplace

In Claude Cowork or Claude Code, run:

```
/plugin marketplace add strategy-labs/strategy-labs-marketplace
```

### 2. Install a Plugin

```
/plugin install brand-content-os@strategy-labs-plugins
```

### 3. Start Using It

Once installed, run `/brand-setup` to create your brand profile through a guided onboarding wizard, or `/brand-import` if you already have a brand document.
